##### Create a social bookmarking site

Social bookmarking should allow users to add, annotate, edit, and share web page bookmarks. Users should be allowed to share, rate and comment bookmarks added by other users.

##### Functional requirements:
When user is registered and signed in on the home page he can see list of bookmarks that he created. From that page he can add, delete, edit and share bookmarks. Bookmark should contain details about title, description, tags, URL and privacy settings.
On the search page user is allowed to search only public bookmarks from other users. Bookmarks can be searched by category, tags, description and user. List of matched bookmarks contains following information: user, link, description, tags, etc. Each list item contain two links. First link opens bookmark URL in new tab. Second link opens bookmark details page.
On the bookmark details page users can import, rate, comment.
Admin user has same functionalities as regular user. He can also create, read, update, delete categories and bookmakrs. Admin user can block and delete regular user.
